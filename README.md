# Smart Study Planner Agent

## 📌 Overview
Smart Study Planner is a Python Command Line Interface (CLI) application built with GitHub Copilot Chat.  
It helps students organize study tasks, set deadlines, and automatically generates a daily schedule with reasoning.  
This project was developed for the **Microsoft Agents League Hackathon** under the *Creative Apps* track.

---

## ✨ Features
- Add study tasks with deadlines and priority levels.
- Store tasks in a JSON file for persistence.
- Generate a suggested daily schedule.
- **Work IQ Integration:** The agent explains *why* each task gets its assigned hours, showing reasoning based on priority and deadlines.
- Simple CLI commands for ease of use.
